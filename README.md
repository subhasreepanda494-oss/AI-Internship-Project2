# Student Performance Prediction (ML Project)

## 📌 Objective
Predict whether a student will pass or fail based on study hours using Machine Learning.

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🚀 Steps
1. Data creation
2. Data visualization
3. Model training (Logistic Regression)
4. Evaluation (Accuracy, Confusion Matrix)
5. Prediction

## ▶️ How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Enter study hours for prediction

## 📊 Output
Model predicts PASS or FAIL based on input hours.
